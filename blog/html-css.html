Do you know the difference between Margin, Border and Padding?

If the answer is no, then read on.

The best way to explain the difference is by using the example as a guide.

The model above shows the structure of the components: margin; border;padding. The content is the element. When you change the value of the width and height, it is the content's width and height that will reflect the changes.

Margin

The margin is the space around an element. If you have used MS word, Google docs, you will know that margins are auto set to the default, however, you can edit the settings. Similarly, this can also be applied in the CSS style attribute.

Border

The border box is the third overlapping box that surrounds the padding box. The purpose of the html border attribute is to set a visible border width for a table.The border value of most html elements is set to zero.

Padding

Padding is the space between the content and the border. The padding is the space between the actual content and the border of the element. It will have any background-color or background image property that has been applied to an element. (source:webpage).

It is easy to get confused between margin versus padding properties in CSS as they both create the space around the element.

The example above indicates the following:

  1. The blue area is the content that is set to 50% width
  2. The area outside the blue area is the block border which extends the border area to include an empty area as it separates the element.
  3. The margin is the transparent space outside the element.
  4. The margin is set to auto- this is the area outside the block area

POINTS TO REMEMBER

  • If you increase the size of an element's border it will INCREASE the distance between the padding box and the margin box.
  • Margins can be set to auto but you cannot se the padding to auto
  • The values of margin can be auto, length or %.
  • The shorthand property is clockwise: margin top; margin right; margin bottom; margin left.
  • In the web design press Ctrl Shift and I will allow you to view/adjust the margin, border or padding.
  • The values of padding can be length or %.
