Do you know the difference between Margin, Border and Padding?

If the answer is no, then read on.

The best way to explain the difference is by using the example below, especially if you are a visual learner.

The margin is the space around an element. If you have used MS word, Google docs, you will know that margins are auto set to the default, but you can change the settings, just by the margins in Html and CSS.

Margin Padding

Padding is the space between the content and the border. The padding is the space between the actual content and the border of the element. It will have any background-color or background image property that has been applied to an element. (source: ___).

Border

The border box is the third overlapping box that surrounds the padding box. The purpose of the html border attribute is to set a visible border width for a table.The border value of most html elements is set to zero.

It is easy to get confused between margin versus padding properties in CSS as they both create the space around the element.

POINTS TO REMEMBER

  • If you increase the size of an element's border it will INCREASE the distance between the padding bos and the margin box.
  • Margins can be set to auto.
  • In padding you cannot set the padding to auto.
  • On CSS input measurements for margins, start with: the top of page margin, right-hand margin, bottom margin, then left-hand margin
